## ✨ Features / අපගේ අංගයන්

* **🎆 360° Live Thorana (සජීවී තොරණ):** Experience a fully immersive 360-degree virtual Thorana using WebVR technology.
* **🏮 Virtual Vesak Kudu (වෙසක් කූඩු):** Beautifully illuminated digital Vesak lanterns with glowing ambient effects.
* **🪔 Interactive CSS Pahana (ඩිජිටල් පහන් පූජාව):** A fully functional, 100% Pure CSS interactive clay lamp. Click to light the flame and see the dynamic lighting effects!
* **✨ Live Background Effects:** Animated fireflies (ආලෝක බිංදු) and a spinning Dharmachakra (ධර්ම චක්‍රය).
* **🎵 Immersive Audio:** Each page features unique, traditional background music to enhance the spiritual experience.
* **📱 Fully Responsive:** Carefully designed to work seamlessly on both Desktop and Mobile devices.

---

## 🛠️ Technologies Used

This project is built using vanilla web technologies without any heavy frameworks, ensuring fast load times and smooth performance:

* **HTML5** (Structure & WebVR integration)
* **CSS3** (Advanced animations, glowing effects, and pure CSS shapes)
* **Vanilla JavaScript** (DOM manipulation and audio/video handling)
* **A-Frame** (For rendering the 360° Video Sphere)
* **Google Fonts** (Using 'Abhaya Libre' for professional Sinhala typography)
